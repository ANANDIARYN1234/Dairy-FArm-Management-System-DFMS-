# dairy-farm-management-system-DFMS-
A dairy farm requires proper management of cattle, milk production, sales, and daily revenue to run smoothly. In many dairy farms, these records are still maintained manually using notebooks and registers. This traditional method makes it difficult to keep accurate records and track daily activities properly.

The Dairy Farm Management System (DFMS) is a web-based system developed to help dairy farm owners and employees manage farm operations in an organized and simple manner. The system allows users to record cattle details, daily milk collection, milk sales, customer payments, and inventory usage in one place. Admin and employees can access the system using their login credentials according to their roles.

DFMS helps reduce paperwork, minimizes mistakes in record keeping, and makes it easier to access farm data whenever required. By using this system, dairy farms can manage their daily work more efficiently and maintain proper records for future reference.
